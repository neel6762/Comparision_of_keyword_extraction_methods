HPG : a tool for presentation generation in WIS Web Information Systems ( WIS ) support the process of retrieving information from sources on the Web and of presenting them as a hypermedia presentation . Most WIS design methodologies focus on the engineering of the abstract navigation ( hyperlinks ) . The actual presentation generation is less supported . Hera is one of the few WIS methodologies that offer a tool for presentation generation ( HPG ) . The HPG transforms RDF data obtained as the result of a query into a Web presentation suited to the user ( in HTML or WML ) .
